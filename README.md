# Truck-scrapper

Tracks **occupancy, pricing, and bookings** for Truck Parking Club listings.
Data is collected automatically every few hours by GitHub Actions — nothing
needs to be running on your side.

## 📊 View the dashboard

**Easiest — the live site (once this is on `main`):**

> https://honorhomebuyers-jpg.github.io/Truck-scrapper/

Bookmark it on your phone or laptop; it refreshes itself with every scrape.

**Or one click in Codespaces:**

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/honorhomebuyers-jpg/Truck-scrapper?quickstart=1)

Opening a Codespace on this repo auto-starts the dashboard and pops open a
browser tab for it — no commands to type.

## How it works

See [`scraper/README.md`](scraper/README.md) for the full details: what gets
recorded, how occupancy and booking evidence are derived, and how to add
listings or whole markets.
